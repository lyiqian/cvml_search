A tool for searching papers published in top venues in computer vision and machine learning based on the arxiv-sanity-lite.

This particular instance indexes papers from CVPR, ICCV, ECCV, ACCV, BMVC, WACV, NeurIPS, and ICML from 2010 onwards

The code for the arxiv-sanity-lite website is open source at Github under arxiv-sanity-lite.

Developed by @karpathy modified by ykotseruba.
